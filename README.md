**Kinetic Zone** is a modern, responsive fitness website designed to showcase a gym's services, gallery, and community. The website aims to provide a clean, interactive, and user-friendly experience for visitors interested in fitness, training programs, and gym membership.

## Project Overview

The **Kinetic Zone website** is a static, multi-page website that includes:

1. **Homepage (index.html)** – Introduces the gym with a hero section, call-to-action, and branding.  
2. **About Page (about.html)** – Highlights the gym's mission, values, and unique features.  
3. **Gallery Page (gallery.html)** – Displays gym images in a responsive masonry-style grid.  
4. **Contact Page (contact.html)** – Placeholder for a contact form or contact information (optional).

The website is fully **responsive**, ensuring it looks great on mobile devices, tablets, and desktops. It uses **modern CSS features**, including Flexbox, Grid, media queries, and custom fonts for a polished design.

---

## Features

### General Features
- **Responsive Design**: Works on all screen sizes.  
- **Fixed Navigation Bar**: Always visible; includes a **hamburger menu** for smaller screens.  
- **Hero Section**: Prominent call-to-action and visually appealing introduction.  
- **Gallery Section**: Masonry layout that adapts to the screen size.  
- **About Section**: Clear, concise presentation of the gym’s mission and values.  
- **Footer**: Social media icons, copyright info, and sticky layout.  

### Interactive Features
- Hover effects for **links, buttons, and logos**.  
- **Hamburger menu toggle** with smooth animation.  
- Smooth scrolling enabled for better UX.  
- Button links redirect to external pages or forms.
